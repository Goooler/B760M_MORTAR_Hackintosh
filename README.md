# B760M_MORTAR_Hackintosh

* Mainboard: MSI B760M MORTAR
* CPU: 13700F
* RAM: 16G * 2
* Boot SSD: SN770
* Graphics: RX5600XT
* WIFI/Bluetooth: 943602CS PCIE-MINI
